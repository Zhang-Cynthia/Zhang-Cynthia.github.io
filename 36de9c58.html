1 基本概念

Docker官方文档

Docker官网下载安装

1.1 镜像（Image）

Docker镜像就像一个特殊的文件系统，可以提供容器运行时所需的程序、库、资源、配置等等，通过这个镜像可以创建多个容器。

1.2 容器（Container）

容器可以被创建、启动、停止、删除、暂停等，Docker利用容器技术，独立运行一个或者一组应用。

1.3 仓库（Repository）

仓库就是存放镜像的地方，每个仓库可以包含多个标签（Tag），每个标签对应一个镜像。通常一个仓库会包含同一个软件的不同版本的镜像，而标签就常用于对应该软件的各个版本，如果不给定标签，将以latest作为标签。

2 常用命令

docker images   #查看所有本地的主机上的镜像
docker search 镜像名 # 搜索镜像

2.1 下载镜像

docker pull 镜像名:版本号    #不写版本号默认下载最新版latest
docker pull ubuntu:24.04  #例如拉取ubuntu镜像

2.2 删除镜像

docker rmi -f 容器id   #-f就是全部删除，后面的条件是根据什么来删除
#删除多个镜像
docker  rmi -f  容器id  容器id 容器id    

# 删除所有的镜像 
docker  rmi -f $(docker images -aq)

2.3 新建容器并启动

docker run [可选参数] image
参数说明：
--name="Name" ： 容器名字，tomcat01、tomcat02，用来区分容器
-d ：后台方式运行                                
-it：使用交互方式运行，进入容器查看内容
-p ：指定容器的端口   -p  8080:8080  
        -p     ip:主机端口:容器端口                                
        -p     主机端口：容器端口（常用方式）
        -p     容器端口                                                     
        容器端口                                                             
-p： 随机指定端口
#启动容器
docker run -it hexo /bin/bash
#从容器钟退回到主机，直接停止容器并退出
exit
#进入当前正在运行的容器
docker exec -it 容器id bashShell
#例如进入hexo容器
docker exec -it hexo /bin/bash
